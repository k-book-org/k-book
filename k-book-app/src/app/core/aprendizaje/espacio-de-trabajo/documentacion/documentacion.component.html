Ciclo repetitivo For

El ciclo for es un ciclo repetitivo que se usa en determinadas ocasiones, por ejemplo, para repetir ciertas instrucciones un numero determinado de veces como en este ejemplo:

iterar x desde 1 hasta 10 {

imprimir ( x );

}

En este caso se ve la variable x, esta variable se denomina variable de control, esta variable es la que tomará los valores en nuestro ciclo, para este caso la variable x tomará los valores desde 1 (el inicio) hasta 10 (el final)

Asimismo se puede usar cuando se desea recorrer una cadena o un array, en el cual se pasa por cada uno de los elementos y ese valor es que le toma la variable de control

iterar letra en "saludo" {

imprimir( letra );

}
